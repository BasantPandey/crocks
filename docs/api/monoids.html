Monoids Guide

Monoids

Each Monoid provides a means to represent a binary operation and is usually locked down to a specific type. These are great when you need to combine a list of values down to one value. In this library, any ADT that provides both anempty and concat function can be used as a Monoid. There are a few of thecrocks that are also monoidial, so be on the look out for those as well. All Monoids work with the point-free functions mconcat,mreduce, mconcatMap andmreduceMap.

All Monoids provide empty and type function on their Constructors as well as the following Instance Functions: inspect, type, valueOf, empty andconcat.
